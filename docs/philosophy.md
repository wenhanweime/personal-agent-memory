# Philosophy

Most AI memory systems optimize for the agent. Personal Agent Memory starts with the human.

The design goal is simple:

> A local folder that tells you what your agents know, what your projects are doing, and where sensitive facts live.

Principles:

- Local first.
- Human-readable first.
- Secrets encrypted by default.
- Markdown over hidden state.
- MCP as an interface, not as the source of truth.
- Hooks automate capture; humans can still inspect and edit.

This project intentionally stays small. It is a ledger, not a hosted brain.
